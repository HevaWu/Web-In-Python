# Web-In-Python
Using Python to parsing data from website  
* Use "nltk" modult to import stopwords, ngrams(combine words together), sent_tokenize(get the sentence)   
* Use "urllib2" to import the browser  
* Use "operator" to import itemgetter(sort our array)  
* Use "BeautifulSoup" to parse HTML  
* Use "sklearn" to import naive_bayes, LogisticRegression, KNeighborsClassifier, accuracy_score, VotingClassifier, MultinomialNB and CountVectorizer
